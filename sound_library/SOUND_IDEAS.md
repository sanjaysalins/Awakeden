# Biblical-times sound catalogue (living backlog)

Candidate sounds for immersive narration. **Keep adding over time.**
- ✓ = already generated, in the library (`index.json` / `clips/`)
- ⭐ = "core" — highest reuse across many episodes; make these first
- everything else = generate on demand when an episode needs it

Reuse scope: nearly all of these are **neutral** (any episode). Mark a clip `specific`
in the library only if it's tied to one story. See `sound_library.py` + memory `sound-library`.
